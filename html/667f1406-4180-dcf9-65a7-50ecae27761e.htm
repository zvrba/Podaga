      FindTValue, TValueTraits Finds a value equivalent to value.
      FirstTValue Moves this to the smallest value in the subtree.
      GetIteratorTValue Creates an iterator for the tree and pushes the root node (if the tree is not empty) on top of the stack. This makes other iterator algorithms operate over the complete tree.
      LastTValue Moves this to the largest value in the subtree.
      NthTValue Sets this to the n'th element in sorted order in the tree.
      PredTValue Moves this to the previous element in sort order.
      SuccTValue Moves this to the next element in sort order.
