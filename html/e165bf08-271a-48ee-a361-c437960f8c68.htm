      Welcome to Podaga library collection

      Podaga is a collection of libraries that I made while exploring various concepts. Use the navigation bar to explore individual libraries.

      Getting started

      Install the library's NuGet package and read its introductory documentation. The assemblies use strong-signing in order to avoid that someone else uses Podaga namespace for nefarious purposes. I keep a copy of the signing key securely stored and can prove ownership. The key is not necessary for building the code on your own, but you will not be able to sign the assemblies.

      Docs directory is a submodule pointing to a private repository where I maintain the source code for documentation. You do not need to check it out to build the libraries.
