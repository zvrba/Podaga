      Remarks

      Implementations of set operations check whether the IEnumerableT argument is actually an instance of JoinableTreeNodeTValue or SetTreeAdapterTValue, TJoin. If so, a more efficient join-based recursive strategy is used. Otherwise, the operation is performed element-wise. In particular, IntersectWith(IEnumerableTValue) allocates temporary storage in size proportional with the result.
