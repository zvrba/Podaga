      TreeAlgorithms Methods

      Methods

      CopyTValue, TValueTraits Copies all nodes in the subtree starting from root. Each node is cloned if the its transient tag is different from transient.
      DeleteTValue, TJoin Removes a value from the tree.
      FindTValue, TValueTraits Finds a value in the tree.
      InsertTValue, TJoin Inserts a value into the tree.
      NthTValue Returns the n'th element in sorted order in the tree. Using this method is more efficient than the equivalent iterator methods.
      RotLTValue, TJoin Single left rotation using this as pivot node.
      RotLLTValue, TJoin Double left rotation using this as pivot node.
      RotRTValue, TJoin Single right rotation using this as pivot node.
      RotRRTValue, TJoin Double right rotation using this as pivot node.
