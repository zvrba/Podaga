      IValueTraitsTValue Methods

      Methods

      Clone Clones value such that the tag part is safe to mutate independently of the original value. Default implementation returns value which is a sufficient implementation for value types.
      CombineTags

      Computes result = left + result + right on the TAG part of TValue.

      A correct implementation obeys monoidal laws with NilTag as the neutral element.

      Compare Compares two values. Must implement a total order.
