      Podaga.JoinableTree Namespace

      This namespace contains types for the joinable tree framework.

      Classes

      CollectionTreeAdapterTValue, TJoin Adapts a joinable tree to ICollectionT.
      DictionaryTreeAdapterTKey, TValue, TJoin Adapts a joinable tree to IDictionaryTKey, TValue and IReadOnlyDictionaryTKey, TValue.
      IteratorAlgorithms Traversal methods on TreeIteratorTValue. Every method in this class leaves a "trail" on the iterator which may be used to continue iteration.
      JoinableTreeNodeTValue Node of a binary search tree; stores a tagged value. Enumerating a node will return its children in in-order traversal.
      PermutationGenerators Data generators for tree tests.
      ReadOnlyListTreeAdapterTValue, TJoin Adapts a joinable tree to IReadOnlyListT. The collection is nevertheless modifiable through the inherited ICollectionT methods.
      SetTreeAdapterTValue, TJoin Adapts a joinable tree to ISetT and IReadOnlySetT.
      TransientSource This class is used as a default source of unique transient values throughout this assembly.
      TreeAdapterExtensions Contains extension methods for creating different "System" collection views from a tree.
      TreeAlgorithms Extension methods on JoinableTreeNodeTValue. These methods allow to treat any tree node as a tree root for further operations.

      Structures

      ModifyStateTValue This struct is used as input to InsertTValue, TJoin(JoinableTreeNodeTValue, ModifyStateTValue) and DeleteTValue, TJoin(JoinableTreeNodeTValue, ModifyStateTValue) methods.
      TreeIteratorTValue

      Provides iterative tree navigation algorithms. At its core, this struct implements a stack, whereas the actual algorithms are implemented as extension methods on IteratorAlgorithms class.

      TreeSectionTValue Describes a "section" of a tree.

      Interfaces

      IAdaptedTreeTValue, TTree Common interface implemented by all adapters to System.Collection.Generic interfaces.
      IAvlJoinTSelf, TValue Implementation of ITreeTraitsTValue for AVL trees.
      ITreeTraitsTValue A joinable tree must provide an implementation of this interface.
      IValueTraitsTValue Any value stored in the tree must implement this interface.
      IWBJoinTSelf, TValue Implementation of ITreeTraitsTValue for WB trees. The balance factor is hard-coded to 1/4. This value is just below the maximum proven in the paper that makes the tree strongly joinable.
